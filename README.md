# 🎯 Number Guessing Game

A simple and interactive **Number Guessing Game** built with **Python**.  
Test your luck and logic by trying to guess the randomly selected number between 1 and 100!  

---

## 📝 Features
- Generates a random number between 1 and 100.  
- User enters guesses until they find the correct number.  
- Gives hints whether your guess is too high or too low.  
- Counts and displays the number of attempts taken.  

---

## 💻 How to Run
1. Make sure you have **Python 3.x** installed.  
2. Clone the repository:  
```bash
cd python-number-guessing-game

⚡ How to Play
Run the game.
Enter a number between 1 and 100 when prompted.
Follow the hints (“Too high!” or “Too low!”) until you guess correctly.
